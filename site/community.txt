+-------------------------------------------------------------+
|                                                             |
|                       FORKS OF KISS                         |
|                                                             |
| * Members of the community have forked KISS to create their |
|   own distributions with differing software, ideas, goals,  |
|   etc.                                                      |
|                                                             |
+-------------------------------------------------------------+
|                                                             |
| * WYVERTUX (https://github.com/wyvertux/wyvertux)           |
|                                                             |
|   Wyvertux is an experimental Linux distribution that aims  |
|   to completely remove GNU utilities (if it's even          |
|   possible) from the Linux build system.                    |
|                                                             |
+-------------------------------------------------------------+
|                                                             |
| * CARBS LINUX (https://carbslinux.org/)                     |
|                                                             |
|   Carbs Linux is an in-development LinuxÂ® distribution with |
|   a suckless mindset. The base system will consist of only  |
|   the necessary programs to create a Linux distribution.    |
|                                                             |

|                                                             |
|                    THEMED REPOSITORIES                      |

|   "theme" and fulfil a singular purpose.                    |
|                                                             |
+-------------------------------------------------------------+
|                                                             |
| * MYWAYLAND (https://github.com/sdsddsd1/mywayland)         |
|                                                             |
|   Wayland and friends.                                      |
|                                                             |
+-------------------------------------------------------------+
|                                                             |
| * KISS-DBUS (https://github.com/periish/kiss-dbus)          |

|   dunst, libnotify, tiramisu and more.                      |
|                                                             |
+-------------------------------------------------------------+
|                                                             |
| * KISS-KDE (https://github.com/dilyn-corner/KISS-kde)       |
|                                                             |
|   A KISS repository for KDE and friends.                    |
|                                                             |
+-------------------------------------------------------------+
|                                                             |
| * KISS-GAMES (https://github.com/sdsddsd1/kiss-games)       |
|                                                             |
|   50+ open source games from a myriad of genres.            |
|                                                             |
+-------------------------------------------------------------+
|                                                             |
| * KISS-LLVM (https://github.com/konimex/kiss-llvm)          |
|                                                             |
|   A kiss-repo structure to build standalone LLVM.           |
|                                                             |
+-------------------------------------------------------------+
|                                                             |
| * KISS-ZFS (https://github.com/jedavies-dev/kiss-zfs)       |
|                                                             |
|   This repo contains KISS build files for ZFS on Linux.     |
|   Package contains userland utilities and kernel modules.   |
|   It also contains the ZFS event daemon (zed).              |
|                                                             |
+-------------------------------------------------------------+
|                                                             |
| * WYVERKISS (https://github.com/wyvertux/wyverkiss)         |

|   KISS goes GNU, we go the opposite.                        |
|                                                             |
+-------------------------------------------------------------+
|                                                             |
| * KISS-FLATPAK (https://github.com/dylanaraps/kiss-flatpak) |

+-------------------------------------------------------------+
|                                                             |
| * KISS-XFCE4 (https://github.com/periish/kiss-xfce4)        |

+-------------------------------------------------------------+
|                                                             |
| * KISS-RICING (https://github.com/periish/kiss-ricing)      |

|                                                             |
|                         SOFTWARE                            |
|                                                             |
| * Software created by users of KISS, many of which are      |

|   A simplistic system-supervisor.                           |
|                                                             |

|                                                             |
| * SHINIT (https://github.com/cemkeylan/shinit)              |

|   It supports acting upon signals.                          |
|                                                             |
