+-------------------------------------------------------------+
|                                                             |
|                       FORKS OF KISS                         |
|                                                             |
| * Members of the community have forked KISS to create their |
|   own distributions with differing software, ideas, goals,  |
|   etc.                                                      |
|                                                             |
+-------------------------------------------------------------+
|                                                             |
| * WYVERTUX  wyvertux/wyvertux)                              |
|                                                             |
|   Wyvertux is an experimental Linux distribution that aims  |
|   to completely remove GNU utilities (if it's even          |
|   possible) from the Linux build system.                    |
|                                                             |
|                                                             |
| * CARBS LINUX (https://carbslinux.org/)                     |
|                                                             |
|   Carbs Linux is an in-development Linux® distribution with |
|   a suckless mindset. The base system will consist of only  |
|   the necessary programs to create a Linux distribution.    |
|                                                             |
+-------------------------------------------------------------+

+-------------------------------------------------------------+
|                                                             |
|                    THEMED REPOSITORIES                      |
|                                                             |
| * The following list of repositories follow a specific      |
|   "theme" and fulfil a singular purpose.                    |
|                                                             |
+-------------------------------------------------------------+
|                                                             |
| * MYWAYLAND  sdsddsd1/mywayland)                            |
|                                                             |
|   Wayland and friends.                                      |
|                                                             |
|                                                             |
| * KISS-DBUS  periish/kiss-dbus)                             |
|                                                             |
|   DBus and software that depends on it. Includes Bluez,     |
|   dunst, libnotify, tiramisu and more.                      |
|                                                             |
|                                                             |
| * KDE  dilyn-corner/KISS-kde)                               |
|                                                             |
|   A KISS repository for KDE and friends.                    |
|                                                             |
|                                                             |
| * GAMES  sdsddsd1/kiss-games)                               |
|                                                             |
|   50+ open source games from a myriad of genres.            |
|                                                             |
|                                                             |
| * LLVM  konimex/kiss-llvm)                                  |
|                                                             |
|   A kiss-repo structure to build standalone LLVM.           |
|                                                             |
|                                                             |
| * ZFS  jedavies-dev/kiss-zfs)                               |
|                                                             |
|   This repo contains KISS build files for ZFS on Linux.     |
|   Package contains userland utilities and kernel modules.   |
|   It also contains the ZFS event daemon (zed).              |
|                                                             |
|                                                             |
| * WYVERKISS  wyvertux/wyverkiss)                            |
|                                                             |
|   Wyverkiss is an alternative rootfs for KISS Linux. If     |
|   KISS goes GNU, we go the opposite.                        |
|                                                             |
|                                                             |
| * FLATPAK  dylanaraps/kiss-flatpak)                         |
|                                                             |
|   Flatpak and friends.                                      |
|                                                             |
|                                                             |
| * XFCE4  periish/kiss-xfce4)                                |
|                                                             |
|   XFCE4 desktop environment for KISS Linux.                 |
|                                                             |
|                                                             |
| * RICING  periish/kiss-ricing)                              |
|                                                             |
|   Tools to help your KISS install look nice.                |
|                                                             |
|                                                             |
| * FONTS  Himmalerin/kiss-fonts)                             |
|                                                             |
|   Fonts packaged for KISS Linux.                            |
|                                                             |
|                                                             |
| * PULSEAUDIO  periish/kiss-pulseaudio)                      |
|                                                             |
|   Pulseaudio and libsndfile for KISS.                       |
|                                                             |

+-------------------------------------------------------------+
|                                                             |
|                         SOFTWARE                            |
|                                                             |
| * Software created by users of KISS, many of which are      |
|   portable to multiple distributions.                       |
|                                                             |
+-------------------------------------------------------------+
|                                                             |
| * TINYRAMFS  illiliti/tinyramfs)                            |
|                                                             |
|   A tiny initramfs generator written in POSIX shell with    |
|    support for LUKS, LVM, mdev, mdevd, udev, etc.           |
|                                                             |
|                                                             |
| * SYSMGR  cemkeylan/sysmgr)                                 |
|                                                             |
|   A simplistic system-supervisor.                           |
|                                                             |
|                                                             |
| * SHINIT  cemkeylan/shinit)                                 |
|                                                             |
|   Basic init daemon in POSIX sh with only 5 lines of code.  |
|   It supports acting upon signals.                          |
|                                                             |
|                                                             |
| * TUITUBE  djt3/tuitube/)                                   |
|                                                             |
|   A minimal tui youtube (invidious) frontend made in c++.   |
|                                                             |
